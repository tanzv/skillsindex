# Language Rules

1. All source code and code comments must be written in English.
2. Chinese must not appear in source code files.
3. Markdown documentation under `docs/` or `user-docs/` may contain Chinese when needed.
4. Historical documents under `agents/records/` may contain Chinese, but new policy/standard files must stay in English.
5. For non-doc files outside those folders, keep content in English.
