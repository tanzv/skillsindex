# 用户端操作指南

## 1. 市场搜索

入口：`/`

### 1.1 关键词模式

1. 选择模式为“关键词”
2. 填写关键词（如 `go api`）
3. 可叠加标签、分类、子分类、排序
4. 点击搜索

### 1.2 AI 语义模式

1. 选择模式为“AI 语义”
2. 输入自然语言问题
3. 点击搜索获取语义匹配结果

## 2. 分类与时间线

### 2.1 分类浏览

入口：`/categories`

1. 点击任意分类进入详情
2. 在分类详情页进行二次筛选

### 2.2 时间线查看

入口：`/timeline`

1. 切换 day/week/month 观察趋势
2. 查看新增量与累计量

## 3. 技能详情与互动

入口：`/skills/{skillID}`

### 3.1 查看详情

可查看：

1. 技能内容
2. 标签、分类、来源
3. 收藏数、评分、评论

### 3.2 收藏

1. 登录后点击“加入收藏/取消收藏”
2. 页面会提示操作成功或失败

### 3.3 评分

1. 选择 1~5 分
2. 提交评分（重复提交会覆盖）

### 3.4 评论

1. 输入评论内容并提交
2. 评论最大长度 3000
3. 可删除本人评论

## 4. 登录与语言切换

1. 登录入口：`/login`
2. 注册入口：`/register`（若系统开启）
3. 语言切换：页面右上角 `EN/ZH`
4. 中文别名入口：`/zh`、`/zh/*`

## 5. 账户中心与个人 API 凭证

### 5.1 账户中心入口

入口：

1. 用户中心下拉菜单中的“账户中心”
2. `/account/profile`

可用分区：

1. 个人资料
2. 安全设置
3. 会话管理
4. API 凭证

### 5.2 创建个人 API 凭证

入口：

1. 用户中心下拉菜单中的“API 凭证”
2. `/account/api-credentials`

操作步骤：

1. 填写凭证名称
2. 填写用途说明
3. 选择过期时间
4. 选择需要的 scopes
5. 点击创建

注意事项：

1. 创建成功后，明文 token 只显示一次
2. 建议按最小权限原则选择 scopes
3. 如仅需公开检索能力，优先选择只读 scopes

### 5.3 轮换、撤销与 scope 更新

在凭证列表中可执行：

1. 更新 scopes
2. 轮换凭证
3. 撤销凭证

建议：

1. 定期轮换长期使用的凭证
2. 发现泄露风险时立即撤销
3. 对 CI、本地脚本、第三方集成分别使用独立凭证
