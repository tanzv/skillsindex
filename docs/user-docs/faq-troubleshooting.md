# 常见问题与排障

## 1. 登录与注册

### Q1：为什么我启动了 `cmd/server` 或 `cmd/api`，却没有默认管理员账号？

- 原因：默认管理员初始化已经从常规服务启动中拆出
- 处理方式：先执行 `go run ./cmd/bootstrap`，再启动 `go run ./cmd/server` 或 `go run ./cmd/api`

### Q2：为什么看不到“创建账号”？

- 可能原因：`ALLOW_REGISTRATION=false`
- 处理方式：联系管理员开启注册，或使用管理员分配账号

### Q3：登录后仍跳转登录页

- 可能原因：会话 Cookie 无效或过期
- 处理方式：清理浏览器 Cookie 后重新登录

## 2. 后台访问

### Q4：为什么我能登录但看不到后台？

- 可能原因：当前角色为 `viewer`
- 处理方式：请 super_admin 调整角色

### Q5：为什么无法修改某条技能？

- 可能原因：你不是该技能 owner，且非 admin/super_admin
- 处理方式：由 owner 或管理员执行

## 3. 导入与同步

### Q6：为什么我启动服务后看不到示例数据？

- 原因：示例数据初始化已经从常规服务启动中拆出
- 处理方式：先执行 `go run ./cmd/bootstrap`

### Q7：Zip 导入失败

排查：

1. 是否为 `.zip`
2. 包内是否有可读内容文件
3. `skill.json` 是否是合法 JSON

### Q8：仓库同步失败

排查：

1. 仓库地址是否可访问
2. 分支是否存在
3. 子路径是否正确
4. 服务节点是否安装 git

### Q9：为什么不能重同步某条技能？

- 仅 `repository` 与 `skillmp` 来源支持重同步

## 4. API Key 与接口

### Q10：接口返回 401

排查：

1. 是否携带 API Key
2. Key 是否已撤销或过期
3. Bearer 格式是否正确

### Q11：OpenAPI 地址打不开

排查：

1. 服务是否正常启动
2. 访问路径是否正确（`/openapi.json` 或 `/docs/openapi.json`）

## 5. 钉钉授权

### Q12：钉钉授权后提示失败

排查：

1. 回调地址是否与配置一致
2. 是否存在跨环境回调错误
3. 授权是否已绑定其他账号

## 6. 审计与安全

### Q13：某些操作报 CSRF 错误

- 原因：缺少或过期 CSRF token
- 处理方式：刷新页面后重新提交表单

### Q14：如何追溯关键操作？

- 进入后台 Audit 分区查看审计日志
