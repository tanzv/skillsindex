# 原型风格审计与导出说明

## 本次优化范围

- 全局深色主题：统一底色、顶栏色、主卡片蓝色层级，减少离散蓝色导致的视觉噪声。
- 全局浅色主题：统一画布底色、顶部栏色、主内容卡片层级（白卡 + 浅底），提升可读性和专业感。
- 异常状态页（浅色）：加载 / 空态 / 错误 / 权限不足卡片统一提升为白底，强调信息层级。
- 主布局节奏：核心页面主内容区间距统一至一致节奏，降低页面间跳变。

## 风格基线（收敛后）

- 深色画布：`#050C18`
- 深色顶栏：`#0B1B37`
- 深色主卡：`#123057` / `#153A72` / `#1A4B88` / `#17406F`
- 浅色画布：`#EAF1FB`
- 浅色顶栏：`#CFDCF1`
- 浅色主层：`#D9E5F8` / `#E6EEFB`
- 浅色卡片：`#FFFFFF` / `#F4F8FF`
- 功能色：保留原有语义（成功/错误/告警）

## 校验结果

- `snapshot_layout(...problemsOnly=true)`：No layout problems.
- 抽样复核：首页、搜索、详情、后台仪表盘、导入中心、同步导出中心、API Key 治理、状态页（深浅）均无越界或错位。
