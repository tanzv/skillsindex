# 背景与目标

## 1. 背景

SkillsIndex 是面向团队的技能资产管理与同步平台，核心问题是将分散在 Markdown、仓库、外部平台（SkillMP）的技能内容统一管理，并通过权限、审计、API 能力进行组织化运营。

现有产品能力已经覆盖：

1. 市场发现（关键词检索、AI 语义检索、分类、时间线）
2. 技能导入（手动、Zip、仓库、SkillMP）
3. 账号与角色（viewer/member/admin/super_admin）
4. 互动系统（收藏、评分、评论）
5. 后台治理（API Key、审计日志、角色管理）
6. 钉钉 OAuth 个人授权

## 2. 业务目标

### G-001 建立统一技能资产入口

- 所有技能内容可被统一索引、检索、浏览、详情查看
- 技能记录具备来源、可见性、分类、标签等元数据

### G-002 建立可控的团队协作机制

- 通过角色能力限制创建、修改、治理边界
- 对关键管理动作保留可追溯审计链路

### G-003 建立可复用的对外能力

- 通过 API Key + OpenAPI 提供检索能力
- 支持后续第三方系统集成（如知识库、工作流引擎）

### G-004 建立可持续同步机制

- 仓库和 SkillMP 来源可执行重同步
- 同步失败不破坏原有稳定数据

## 3. 成功标准（业务层）

1. 用户可在市场页快速定位目标技能（关键词、标签、分类）
2. 运营可通过后台完成全量技能治理与审计复盘
3. 开发者可用 API Key 调用公开检索接口并对齐 OpenAPI
4. 钉钉授权用户可成功查询个人资料接口 `/api/v1/dingtalk/me`
